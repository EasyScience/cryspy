property neutron_magnetic_j0_a1¶

First, the parameters are used directly to approximate spatial averages of spherical Bessel functions over the electronic wave functions of unpaired electrons of the given atom type as a function of s = sin(θ)/λ.

\[\left\langle j_{n}(s)\right\rangle = \left( A_{1} e^{−a_{2}s^{2}} + B_{1} e^{−b_{2}s^{2}} + C_{1} e^{−c_{2}s^{2}} + D \right) \times \left[\left. 1 \right|_{n = 0} \vee \left.s^{2}\right|_{n = 2, 4, 6}\right]\]

The \(\left\langle j_{n}(s)\right\rangle\) are then combined to determine the spin and orbital contributions to the magnetic form factor of the atom. The ‘e’ parameter is a measure of error in the approximation.
