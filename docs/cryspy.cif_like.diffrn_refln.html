DiffrnRefln¶

class cryspy.cif_like.cl_diffrn_refln.DiffrnRefln(index_h=None, index_k=None, index_l=None, fr=None, fr_sigma=None, intensity_up_calc=None, intensity_down_calc=None)[source]¶

Bases: cryspy.common.cl_item_constr.ItemConstr

Data items in the DIFFRN_REFLN category record details about the intensities measured in the diffraction experiment.

The DIFFRN_REFLN data items refer to individual intensity measurements and must be included in looped lists.
